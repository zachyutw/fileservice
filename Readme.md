# NodeFileService

A Node file service supports multiple real world use cases.

# Keywords

| express, typescript

# Features

-   normal files upload, and api support
-   image upload support
-   resize image support
-   water mark support
-   userId support
-   cors support
-   swagger support
-   ejs support
